# Blocos de códigos

# Exemplo em linha

Informe os parâmetros `username` e `password` para a função `login()`.

# Exemplo em bloco 1

        // login.js

        const username = 'joseneto';
        const password = 'secret';

        login(username, password);

# Exemplo em bloco 2

```
// login.js

const username = 'joseneto';
const password = 'secret'
login(username, password);
```
